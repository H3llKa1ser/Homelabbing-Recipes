## 🆓 Free Structured Learning Path (No Cost at All)

If you want to **spend zero money** and still prepare effectively, follow this path:

### Month 1 – Foundation Concepts
```
Week 1:
□ Register free at opengroup.org
□ Read TOGAF Standard Introduction & Part I (Architecture Capability)
□ Watch: "TOGAF Overview" on The Open Group YouTube channel
□ Tool: Install Archi (free) and explore the interface

Week 2:
□ Read TOGAF Part II – Architecture Development Method (ADM)
□ Focus: Preliminary Phase + Phase A (Architecture Vision)
□ YouTube: Search "TOGAF ADM Phase A explained"
□ Exercise: Write an Architecture Vision for one of the case studies
  we discussed (e.g., MedBank banking transformation)

Week 3:
□ Read ADM Phases B, C, D (Business, Information Systems, Technology)
□ YouTube: Watch walkthroughs of each phase
□ Exercise: Create a simple context diagram using Draw.io for MedBank
□ Document: Write one ADR based on a technology decision in MedBank

Week 4:
□ Read ADM Phases E, F, G, H (Solutions, Migration, Governance, Change)
□ Focus on Architecture Governance — very exam-heavy
□ Exercise: Create a simple Architecture Roadmap for MedBank
□ Revision: Summarise each ADM phase in one paragraph from memory
```

### Month 2 – Deep Dive & Application
```
Week 5:
□ Read TOGAF Part III – ADM Guidelines & Techniques
□ Focus: Stakeholder Management, Architecture Principles,
  Gap Analysis, Migration Planning
□ Exercise: Define 5 Architecture Principles for MedBank

Week 6:
□ Read TOGAF Part IV – Architecture Content Framework
□ Understand: Artefacts, Deliverables, and Building Blocks
□ Exercise: Map TOGAF deliverables to ADM phases in a table
□ Tool: Build a simple ArchiMate model in Archi

Week 7:
□ Read TOGAF Part V – Enterprise Continuum & Tools
□ Read TOGAF Part VI – TOGAF Reference Models (TRM, III-RM)
□ Exercise: Apply Gap Analysis to the NationalHealth case study

□ Read TOGAF Part VII – Architecture Capability Framework

```
Week 9:
□ Full revision of all ADM phases — draw the cycle from memory

□ Focus on areas where you scored below 70%
□ Flashcards: Create cards for key TOGAF terms and definitions

Week 10:
□ Practitioner-level preparation — scenario-based questions
□ Practice applying ADM to new scenarios you haven't seen before

Week 11:
□ Full mock exam (timed) — Foundation level
□ Full mock exam (timed) — Practitioner level
□ Review every wrong answer in detail
□ Re-read any ADM phases where you struggled

Week 12:

## 💡 Final Tips

- 📖 **Read the official standard first** — even partially. It trains your brain to think in TOGAF language, which directly helps in exams
- 🔁 **Apply as you learn** — use the MedBank, NationalHealth, and RetailMax case studies we built together to practise each ADM phase as you study it
- 🧠 **Don't just memorise** — TOGAF exams (especially Practitioner) test application and judgement, not just recall
- 📝 **Take notes in your own words** — then compare back to the standard. This is the most effective revision technique
- 🎯 **Target 80%+ in mock exams** before booking the real thing — the pass mark is 55% for Foundation and 60% for Practitioner, so 80% in mocks gives you a solid buffer

> 🚀 **You can be fully exam-ready for TOGAF in 3 months spending little to nothing.** The standard is free, the tools are free, and YouTube is free. The only thing required is your time and consistency.
