# VPN Server

### Specs

 - OS: Debian

 - RAM: 1024MB 

 - Storage Space: 15GB

#### 1) Installation

Download Debian .iso file from:

    https://www.debian.org/download

Then, create a new VM in VirtualBox Manager and set the specs mentioned above.

#### 2) Configuration

Login as root (enter your password)

    su root
